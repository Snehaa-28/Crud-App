# Django Todo CRUD App

A **web-based Todo List application** built with Django that allows users to **Create, Read, Update, and Delete (CRUD)** tasks easily through a clean and responsive interface.
The project demonstrates backend development using Django Framework and frontend design using HTML and CSS(Bootstrap).

## Features
- User Authentication (Register & Login)
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive and interactive dashboard
